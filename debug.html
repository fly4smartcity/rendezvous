About Rendezvous

Rendezvous is a simple web-based GUI that runs in your browser and displays the position of your device/robot on a global map in real time.

RV supports the ROS standard messages and has been created by:

  • Gabriele Ermacora
  • Stefano Rosa
  • Antonio Toma
feel free to contact us for info and/or if you want to actively contribute to the project.

Randezvous uses the following third part libraries:

  • Bootstrap
  • jQuery
  • RosLibjs
  • Bootstrap Flat UI
  • Flight Indicators
